# iDesk 使用教程

## 总览

iDesk 是一个桌面效率工具，由 **桌面组件(Widget)** 和 **临时窗口(Transient)** 两大部分组成。

桌面组件可以贴在桌面上，拖到屏幕边缘会自动变成**书签**；临时窗口通过快捷键呼出，用完即走。

---

## 基本概念

### 控制台

按下 `Alt + W` 打开主控制台。控制台是管理所有组件的中心，你可以在这里创建/删除各类组件实例。

### 桌面组件（Widget）

通过控制台创建的**待办**、**习惯**、**日程**、**笔记**等会以独立窗口的形式在桌面上显示。

- **创建方式**：控制台 → 点击对应模块的「创建实例」按钮
- **拖动**：鼠标左键按住窗口顶部区域可以拖动
- **书签化**：将窗口拖到屏幕左/右边缘，自动收缩为书签
- **书签样式切换**：窗口左上角的 ◈ 按钮可以切换书签的样式
- **最小化到书签**：窗口右上角的 ─ 按钮可以将窗口最小化为书签
- **恢复**：点击书签即可恢复为正常窗口

### 临时窗口（Transient）

通过快捷键呼出的浮动窗口，用完后自动关闭或手动关闭。

- **Chat（AI 聊天）**：`Alt + Space`
- **翻译**：`Alt + C`
- **截图识别 → Chat**：`Alt + Shift + S`
- **截图识别 → Translate**：`Alt + Shift + C`

---

## 快捷键一览

- Alt + W => 打开/隐藏主控制台
- Alt + S => 显示/隐藏所有桌面组件
- Alt + X => 所有组件置顶闪现（定位用）
- Alt + V => 将所有组件最小化到书签
- Alt + Space => 打开 AI 聊天窗口
- Alt + C => 打开翻译窗口
- Alt + Shift + S => 截图识别文字 → 发送到 Chat
- Alt + Shift + C => 截图识别文字 → 发送到 Translate
- F3 => 打开调试页面(显示上一次发送的 prompt)

> 所有快捷键均可在控制台 → 设置 → 按键绑定中自定义修改。

## 各组件用法

### 待办（Todo）

分组管理待办事项。

- 每个分组是一个独立的桌面组件
- 支持设置截止日期（DDL）
- AI 聊天中可以查询和操作待办

### 习惯（Habit）

每日打卡的习惯追踪器。

- 周视图展示一周的打卡情况
- 点击圆点切换打卡状态
- AI 可以查询打卡情况和执行打卡

### 日程（Schedule）

日历形式查看和管理日程。

- 按天或范围查看日程
- AI 可以查询、添加、删除日程

### 笔记（Notes）

简单的记事本。

- 每个笔记是一个独立的桌面组件
- 点击标题可以编辑标题
- 正文区域支持多行文本

### 计划（Plan）

今日计划管理 + AI 规划助手。

- 左侧：今日计划列表，支持添加/完成/删除
- 右侧：AI 规划助手，可以帮你制定今日计划
- 点击日期标签可以跳转日期
- 核算今日：自动对比完成情况

### 翻译（Translate）

AI 驱动的翻译工具。

- 输入文本后自动识别语种并翻译
- 支持逐词释义、例句、同义词等

## AI 聊天

AI 聊天是 iDesk 的核心功能之一，可以在临时窗口（`Alt + Space`）或控制台中使用。

### API 配置

首次使用需要配置 API 密钥：

1. 打开控制台 → **设置**
2. 在 **LLM 设置** 区域选择 API 格式（OpenAI / Anthropic）
3. 填写 **请求地址** 和 **请求密钥**
4. 点击 **保存**

支持兼容 OpenAI 或 Anthropic 格式的任意 API 服务。

### 功能

- 自然语言操作待办、习惯、日程、计划
- 多轮工具调用（AI 可以连续执行多个操作）
- Lite 模式：纯文本对话，不调用工具
- 上下文压缩：长对话自动压缩历史

## 书签样式

所有桌面组件都支持书签模式。点击左上角 ◈ 按钮可以浏览和切换书签样式。

目前提供四类样式：

- **默认**：经典蓝色双竖线
- **特殊形状**：旗帜、胶囊、六边形等共 10 种异形书签
- **纯色系列**：12 种纯色书签
- **深色系列**：10 种深色书签

选择后重启组件即可生效。
